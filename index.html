Accessibility of Bike Networks in Amsterdam

What are the benefits to creating new bike infrastructure in Amsterdam? This project set out to determine what the impact would be between different bike network options in the Metropolitan Region of Amsterdam. For this work we cooperated with the municipality to identify and model missing bike links in the region. In addition, we analyzed the current bike network and determined where upgrades were necessary for a more safe and efficient regional bike network.

This analysis contains three models: Reference, Model3 and Speed PedelEc. The Reference Model, is the network as it currently is with no changes. Model 3 is the current network with identified missing links added and speeds increased on regional and urban ‘bike highways’ to simulate upgrades to the network. The last model tested, Speed PedelEc (SP), is the same as Model 3, but with upgrades on specific links in the network that would allow 35 km/h travel via electric bikes.

Each dot represents a starting point in the city and clicking on it reveals how far you can travel in 45 minutes via each model. Number of jobs reached are used as the variable to calculate accessibility.
